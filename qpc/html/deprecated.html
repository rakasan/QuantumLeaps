Global QFsm
QFsm state machine; instead use: QHsm. Legacy state machines coded in the "QFsm-style" will continue to work, but will use the QHsm implementation internally. There is no longer any efficiency advantage in using the "QFsm-style" state machines.

QFsm state machine constructor; instead use: QHsm_ctor()
